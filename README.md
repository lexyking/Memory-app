# Memory app

This project let you record, update and delete the places, articles, videos that you have read and found interesting.

## How to run locally

Client:

- npm install
- npm start

API:

- npm install
- nodemon index.js

## Technologies

Client:

- React
- Moment
- Axios
- Redux and Redux thunk

API:

- ExpressJs
- Cors
- Body-parser
